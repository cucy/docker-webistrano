docker-webistrano
=================

## Usage

```bash
docker run --name my-webistrano --link some-mariadb:mysql -d pomu0326/webistrano
```

## Environment Variables

### `MYSQL_HOST`

MySQL Database Host's IP Address or HostName.

### `MYSQL_DATABASE`

MySQL Database Name.

### `MYSQL_USER`

MySQL User Name.

### `MYSQL_PASSWORD`

MySQL User Password.
